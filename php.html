    • Variables

    • Los nombres de funciones y clases definidas por el usuario son case-insensitive (ej. echo, Echo, ECHO ). Por su parte, las variables son case-sensitive (ej. $var, $Var, $VAR son diferentes).
    • En PHP los enunciados se separan por un punto y coma, excepto cuando se trata de la última línea de código antes de las etiquetas de cierre de PHP "?>"". <?php /* some code */ ... echo "Última línea" //No semicolon required ?>
    • Existen tres formas de agregar comentarios en PHP: # single line comment... // single line comment... /* Multiline comment... */
    • Las variables en PHP se declaran con el símbolo $. El primer carácter debe de ser una letra (incluyendo con acento), un guión bajo "_" o cualquier carácter ASCII del 0x7F (127) al 0xFF (255). Por ejemplo: $name, $_username, $unaEñe, $árbol
    • Una constante es un identificador para un valor que nunca va a cambiar. 
                              
                            define("PI", 3.1416);
                            echo PI; //3.1416

    Tipos de Datos

    • PHP ofrece 8 tipos de datos. Cuatro de ellos son escalares (valor simples): enteros, números flotantes, cadenas de caracteres y booleanos. Dos son especiales (colecciones): arreglos y objetos. Los dos restantes son especiales: recursos (resource) y NULL.
    • Los integers representan números enteros (ej. 1, 20, 987). El tamaño y valor máximo que puede almacenarse como un número entero depende de la máquina que ejecuta PHP. No obstante, el tamaño en bytes del tipo integer de PHP está contenido en la constante PHP_INT_SIZE. El máximo valor aceptado por el tipo integer de PHP está contenido en PHP_INT_MAX. Los integer pueden ser representados como números octales, hexadecimales o binarios. A partir de PHP 7.4 las cantidades pueden contener guiones. Por ejemplo: 3_489.25 // 3489.25
    • Los números octales se escriben con "0" al inicio seguido por cualquier cantidad de dígitos (0-7). Al igual que los numeros decimales, pueden ser positivos o negativos. Por ejemplo 0755 // decimal 493, +010 // decimal 8
    • Los números hexadecimales inician con "0x" seguido por cualquier cantidad de dígitos (0-9) y letras (A-F). Las letras pueden ser mayúsculas o minúsculas. Por ejemplo: 0xFF // decimal 255, -0x10 // decimal -16
    • Los números binarios inician con "0b" seguido por cualquier cantidad de dígitos (0-1). Por ejemplo: 0B10 // decimal 2, 0b11111111 // decimal 255
    • Los números float sirven para representar números con decimales. PHP permite manejar números flotantes como tradicionalmente se hace (ej. 3.1416, 2.5, 9.2) y números con representación exponencial colocando al final una "E" y seguido la potencia de 10 que se desea usar. (ej. 0.314E1 // 0.314 * 10 ^ 1 = 3.14,... 314E-2 // 314 * 10 ^ -2 = 3.14 )
    • Los números de punto flotante no son precisos. Por ejemplo, en muchos sistemas 3.5 en realidad está representado como 3.4999999999. La forma correcta y segura de comparar cantidades flotantes es comparar varios decimales: 
                              
                            if (intval($a * 1000) == intval($b * 1000)) { 
                                // numbers equal to three decimal places
                            }
      El manual de PHP sugiere comparar cantidades flotantes como sigue: 
                              
                            $presicion = 5; //dígitos decimales a comparar
                            $epsilon = 1 * pow(10, -$presicion); // 0.00001 en este caso
                            $a = 1.4;
                            $b = (1.1 + 0.3); // 1.4
                            
                            if ( abs($a - $b) < $epsilon ){ //Realizar la comparación
                                echo "son iguales"; //En este ejemplo son iguales
                            } else {
                                echo "no son iguales";
                            }
